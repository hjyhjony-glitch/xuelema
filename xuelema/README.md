## 功能特性

### P0核心功能（第一批）

1. **任务管理**
   - 创建、编辑、删除任务
   - 任务优先级设置（高/中/低）
   - 任务筛选（全部/今日/本周/已完成）
   - 任务搜索

2. **番茄专注**
   - 25分钟专注 + 5分钟休息
   - 专注期间锁定功能
   - 番茄钟计数显示
   - 进度圆环动画

3. **首页**
   - 今日任务展示
   - 学习统计卡片
   - 功能入口网格

4. **本地存储**
   - SharedPreferences持久化
   - 任务数据本地保存

### 主题
- 清新绿色主题 (#4CAF50)
- Material Design 3设计

## 国际化
- 支持中文 (zh)
- 支持英文 (en)
