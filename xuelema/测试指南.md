# 学了吗APP 测试指南

## 一、环境准备

### 1.1 检查Flutter环境
```powershell
# 检查Flutter版本
flutter --version

# 检查依赖
flutter doctor
```

### 1.2 进入项目目录
```powershell
cd "E:\OpenClaw_Workspace\memory\projects\学了吗APP"
```

---

## 二、安装依赖

```powershell
# 安装依赖
flutter pub get

# 生成国际化文件
flutter gen-l10n

# 运行代码分析
flutter analyze
```

---

## 三、运行测试

### 3.1 连接设备
确保手机已连接（USB调试开启）或者使用模拟器

### 3.2 启动测试
```powershell
# 运行APP
flutter run
```

## 八、常见问题

### 8.1 依赖安装失败
```bash
# 清理缓存重新安装
flutter clean
flutter pub get
```

### 8.2 设备连接失败
```bash
# 检查设备
adb devices

# 重启ADB
adb kill-server
adb start-server
```

### 8.3 构建失败
```bash
# 分析问题
flutter analyze

# 查看详细错误
flutter run --verbose
```
