# 学了吗APP - 定时功能使用指南

## 🎯 快速开始

### 1. 启动应用
- 打开APP后，定时功能自动初始化
- 控制台会显示"通知服务初始化完成"
- 自动设置每日默认提醒

### 2. 查看定时功能
- 进入 **个人中心**（右下角头像图标）
- 点击 **定时提醒设置**
- 可查看和管理所有定时功能

## 📅 每日提醒

### 默认设置
- **早上8:00**: "学习时间到！" - 开始今日学习计划
- **晚上8:00**: "复习时间" - 复习今日学习内容
- **晚上10:00**: "睡前总结" - 总结成果，计划明天

### 自定义设置
1. 进入 **定时提醒设置**
2. 开关每日提醒项
3. 点击 **保存设置** 生效

## 📝 任务提醒

### 自动提醒
- **创建任务时**: 自动安排截止提醒
- **默认提前**: 30分钟提醒
- **任务完成**: 自动取消提醒

### 自定义提前时间
1. 进入 **定时提醒设置**
2. 找到"任务提醒设置"
3. 使用滑块调整提前时间（5-120分钟）
4. 点击 **保存设置**

## ⏱️ 专注计时器

### 使用方法
1. 点击底部导航栏的 **计时器图标**
2. 选择专注时长（15/25/30/45/60分钟）
3. 点击 **开始** 按钮
4. 专注完成后会有通知提醒

### 专注模式
- **专注时段**: 倒计时进行中
- **休息时段**: 专注完成后自动进入
- **完成统计**: 记录专注次数和时间

## 🔄 复习提醒

### 设置复习时间
1. 进入 **定时提醒设置**
2. 找到"复习提醒设置"
3. 开关复习提醒
4. 点击"提醒时间"选择具体时间
5. 点击 **保存设置**

## 🧪 功能测试

### 测试通知
1. 进入 **定时提醒设置**
2. 点击 **测试通知** 按钮
3. 查看手机通知栏

### 查看计划
1. 进入 **定时提醒设置**
2. 点击 **查看计划** 按钮
3. 查看所有已安排的提醒

## ⚙️ 高级功能

### 取消所有提醒
- 点击 **取消所有** 按钮
- 会清除所有已安排的提醒

### 恢复默认设置
- 点击 **恢复默认** 按钮
- 恢复系统预设的提醒时间

### 专注统计查看
- 专注屏幕底部显示统计
- 包括连续学习天数、总专注时间等

## 🔧 故障排除

### 没有收到通知？
1. 检查手机 **通知权限** 是否开启
2. 进入设置 → 应用 → 学了吗 → 通知权限
3. 使用 **测试通知** 验证功能

### 提醒时间不准？
1. 确认手机时区设置正确
2. 重启应用重新初始化
3. 重新保存提醒设置

### 专注计时器不工作？
1. 确保应用在前台运行
2. 检查手机休眠设置
3. 重新启动计时器

## 📱 使用建议

### 最佳实践
1. **早上8:00**: 规划今日学习任务
2. **专注时段**: 每次25分钟，专注学习
3. **晚上8:00**: 复习今日学习内容
4. **晚上10:00**: 总结并计划明日

### 学习节奏
- 专注25分钟 → 休息5分钟
- 完成4个番茄钟 → 长休息15-30分钟
- 每日固定时间复习错题
