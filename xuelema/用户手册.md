# 学了吗APP - 用户手册

## 一、APP简介

### 1.1 产品概述
**学了吗APP** 是一款全学段精准学习辅助工具，帮助学生解决拖延症、知识点遗漏、缺乏规划、不会复盘等问题。

### 1.2 核心功能
| 功能 | 说明 |
|------|------|
| 拖延治愈系统 | 任务拆解、番茄专注、打卡监督、心理疏导 |
| 知识点补漏系统 | 全学段知识点库、自测、错题本、思维导图 |
| 科学规划模块 | 目标设置、计划生成、动态调整 |
| 高效复盘模块 | 每日/每周/每月复盘 |

### 1.3 目标用户
- 中小学生（4年级-高中）
- 大学生
- 考证人群（公考、教资、会计等）

## 三、快速入门

### 3.1 首次使用

1. **打开APP**
   - 首次打开会有引导页
   - 选择学习阶段（小学/中学/大学/考证）

2. **设置目标**
   - 设置短期目标（如"一周复习完数学第二单元"）
   - 设置长期目标（如"月考数学提10分"）

3. **添加任务**
   - 点击"+"添加学习任务
   - 设置截止时间和提醒

## 四、功能使用

### 4.1 任务管理

#### 添加任务
1. 点击首页右下角"+"
2. 输入任务名称
3. 设置截止时间
4. 设置提醒时间
5. 点击"保存"

#### 任务拆解
1. 点击任务进入详情
2. 点击"拆解任务"
3. 输入子任务
4. 设置每个子任务的时间

#### 任务筛选
- **全部任务**：查看所有任务
- **今日任务**：只看今天的任务
- **本周任务**：查看本周任务
- **已完成**：查看已完成的任务

### 4.2 番茄专注

#### 开始专注
1. 点击首页"专注学习"
2. 设置专注时长（默认25分钟）
3. 点击"开始专注"
4. 专注期间手机将被锁定

#### 专注模式
- **普通模式**：专注期间可退出
- **强制模式**：无法退出，必须完成

#### 休息提醒
- 专注完成后自动进入休息（默认5分钟）
- 休息完成后可继续下一个番茄钟

### 4.3 知识点补漏

#### 浏览知识点
1. 点击"知识点补漏"
2. 选择学段（小学/中学/大学）
3. 选择学科
4. 选择章节
5. 查看知识点详情

#### 知识点搜索
1. 在知识点库页面顶部搜索框
2. 输入关键词（如"函数单调性"）
3. 点击搜索
4. 查看搜索结果

#### 自测功能
1. 选择知识点
2. 点击"开始自测"
3. 完成5-10道自测题
4. 查看结果和解析

#### 错题本
1. 点击"错题本"
2. 查看所有错题
3. 点击错题查看解析
4. 设置错题复盘提醒

### 4.4 科学规划

#### 设置目标
1. 点击"规划"
2. 点击"设置目标"
3. 选择目标类型（短期/长期）
4. 输入目标内容
5. 设置截止时间
6. 点击"保存"

#### 查看计划
1. 在规划页面
2. 查看每日/每周计划
3. 点击计划查看详情
4. 标记完成或调整

### 4.5 高效复盘

#### 每日复盘
1. 点击"复盘"
2. 点击"每日复盘"
3. 填写复盘模板
4. 点击"保存"

#### 复盘模板
```
今日学会了哪些知识点？
今日拖延原因是什么？
明日改进方向是什么？
```

#### 查看历史复盘
1. 点击"复盘"
2. 点击"历史复盘"
3. 查看过往复盘记录

### 4.6 个人中心

#### 查看统计数据
1. 点击"我的"
2. 查看学习统计
3. 查看连续打卡天数
4. 查看专注时长

#### 设置
1. 点击"设置"
2. 可设置：
   - 主题模式（浅色/深色/跟随系统）
   - 专注时长
   - 提醒时间
   - 语言（中文/English）

## 六、常见问题

### Q1: 专注模式无法退出？
A: 这是正常现象。专注模式就是为了防止分心。如果需要退出，可以在设置中关闭"强制专注"。

### Q2: 知识点搜索不到？
A: 请检查关键词是否正确，或尝试更宽泛的搜索词。

### Q3: 打卡失败？
A: 请检查网络连接，或稍后重试。

### Q4: 如何删除任务？
A: 在任务列表长按任务，选择"删除"。

### Q5: 数据会丢失吗？
A: 数据存储在本地，建议定期备份。
